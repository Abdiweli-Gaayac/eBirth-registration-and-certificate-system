# Online Birth Registration & Certificate System

A frontend-only React application for birth registration using React.js and Tailwind CSS.

## Features

- Birth Registration Form with:
  - Child Information (Full Name, Gender, Date of Birth, Place of Birth)
  - Father Information (Full Name, Phone Number, National ID)
  - Mother Information (Full Name, Phone Number, National ID)
- Professional government-style layout
- Responsive design
- Controlled form inputs

## Installation

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```

## Tech Stack

- React 18
- Tailwind CSS 3
- Vite
